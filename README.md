🛒 E-Commerce Website (Frontend Project)

A fully responsive E-Commerce Web Application built using HTML, CSS, and JavaScript. This project simulates a complete online shopping workflow including authentication pages, product browsing, cart management, and user settings — all built without any backend framework.

Project Overview

This project demonstrates core frontend development skills by implementing: User authentication UI (Login & Signup) Product listing and browsing Shopping cart functionality Settings management page Responsive UI using pure CSS Dynamic behavior using Vanilla JavaScript The application is designed to mimic a real-world e-commerce platform interface.

Tech Stack

HTML5 – Page structure CSS3 – Styling & responsive design JavaScript (Vanilla JS) – Dynamic functionality Git & GitHub – Version control

Features

Home page with navigation 🛍️ Product listing interface 🛒 Add to cart functionality ➕ Increase / Decrease product quantity ❌ Remove items from cart 💰 Dynamic total price calculation 🔐 Login & Signup UI ⚙️ Settings page UI 📱 Fully responsive design
